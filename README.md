# Mood-Based-Music-Recommender
This project recommends different kinds of  music based on the mood of the listener and previous listening patterns of audience.
The Mood-Based Music Recommender is a simple web application that suggests music based on the listener’s current mood.
It helps users quickly find music that matches how they feel, improving their listening experience.

Live Demo:
https://sammchita.github.io/Mood-Based-Music-Recommender/

It works in following way:
•	The user selects a mood (happy, sad, energetic, relaxed, etc.)
•	The app recommends music tracks/playlists related to that mood
•	Based on users mood it suggests links of songs to user
•	Recommendations are based on predefined mood–music mapping and basic user interaction patterns

 Technologies Used
•	HTML – structure of the app
•	CSS – styling and layout
•	JavaScript – mood selection logic and recommendations

Features
•	Simple and clean user interface
•	Mood-based music suggestions
•	Fast and lightweight (no backend required)
•	Works directly in the browser

Project Purpose
This project was built to:
•	Practice JavaScript logic
•	Understand user-based recommendations
•	Strengthen frontend fundamentals
•	Build a beginner-friendly portfolio project
